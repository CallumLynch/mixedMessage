*Mixed Messages Scope*

**Requirements**
+ Output needs to have 3 pieces of input data
+ When program is run, a random message should be generated

**My Scope**

A random message that pulls 3 pieces of data (scenario, size and animal) e.g. (I was (walking home from school) when i was attacked by a (tiny) (badger).).

It's going to need;

3 arrays, 1 for each piece of data needed for the sentence
A random number generator to figure out which array to select
All three selected arrays need to then be pulled into one location and returned to the console.
